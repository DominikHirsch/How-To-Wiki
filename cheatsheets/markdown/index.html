    fett

    Listen

    Ungeordnete Listen

    Syntax

    - Punkt 1
- Punkt 2
  - Unterpunkt 2a
  - Unterpunkt 2b
* Sternchen statt Bindestrich

    Ergebnis

    • Punkt 1

    • Punkt 2

      • Unterpunkt 2a
      • Unterpunkt 2b
    • Sternchen statt Bindestrich

    Geordnete Listen

    Syntax

    1. Erster
2. Zweiter
   1. Unterpunkt a
   2. Unterpunkt b

    Ergebnis

    1. Erster

    2. Zweiter

      1. Unterpunkt a
      2. Unterpunkt b
